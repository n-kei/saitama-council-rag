Nano Banana
is the name for Gemini's native image generation capabilities. It currently refers to two distinct models available in the Gemini API:
Nano Banana
: The
Gemini 2.5 Flash Image
model (
gemini-2.5-flash-image
). This model is designed for speed and efficiency, optimized for high-volume, low-latency tasks.
Nano Banana Pro
: The
Gemini 3 Pro Image Preview
model (
gemini-3-pro-image-preview
). This model is designed for professional asset production, utilizing advanced reasoning ("Thinking") to follow complex instructions and render high-fidelity text.
Get started
You can generate images using the
generate_content
method using the model name that corresponds to the version you'd like to use.

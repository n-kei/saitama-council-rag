Resource
Field
A ResourceField is a basic building block of a
Dynamic
Resource
Definition
, each resource
field corresponds to a output variable of the current step. A single
Dynamic
Resource
Definition
can contain multiple resource fields.
Only available for Google Workspace add-ons that extend Google Workspace Studio.
